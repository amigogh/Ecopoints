# EcoPoints - Waste-to-Rewards Recycling System

A demo presentation app for an EcoPoints waste-to-rewards recycling system that operates through vendor machines.

## Features

- **OTP Verification System**: Secure authentication via mobile OTP
- **Points Tracking**: Real-time tracking of earned eco-points
- **Rewards Catalog**: Redeem points for:
  - Mobile balance recharge (Grameenphone, Robi, Banglalink, Teletalk)
  - Cash rewards (bKash, Nagad)
  - Discount coupons (KFC, supermarkets, ride-sharing, e-commerce)
- **Account Management**: Connect mobile numbers and mobile banking accounts
- **Transaction History**: View all earned and redeemed points

## Local Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Deployment

This app is configured for easy deployment on Vercel. Simply connect your repository and deploy!
